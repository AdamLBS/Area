
# Area

A backend, web app and mobile app project.

## Web app

For test:
- ```yarn```
- ```yarn dev```

For build: ```yarn build```

## Mobile app

In order to build the Flutter app :

First get dependencies :
- ```flutter pub get```
Then launch an android emulator & run this command :
- ```flutter run```

In order to create a .apk :
```flutter build apk```
